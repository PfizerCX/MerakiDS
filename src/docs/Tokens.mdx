# Design Tokens

Meraki Design System tokens are the foundational values that drive every component. They are authored as CSS custom properties in `tokens.css` and can be consumed directly via `var(--token-name)` or through the TypeScript token constants.

```css
/* Usage in CSS */
.my-element {
  color: var(--color-primary);
  padding: var(--spacing-16);
  border-radius: var(--radius-2);
}
```

```tsx
// Usage in TypeScript
import { colors, spacing } from '@meraki-ds/react/tokens';

const style = { color: colors.primary, padding: spacing['16'] };
```

## Typography

Font family: **Pfizer Diatype** (sans), **Pfizer Diatype Mono** (code), **Pfizer Tomorrow** (display alternate).

All type styles use a line-height of **1.2** and letter-spacing of **0px** (regular) or **-1px** (tight, for headlines/display).
